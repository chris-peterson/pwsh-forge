# Get-Issue

## SYNOPSIS

List or get issues

## SYNTAX

### __AllParameterSets

```
Get-Issue [[-Id] <string>] [-State <string>] [-Mine] [-Group <string>] [-Assignee <string>]
 [-Author <string>] [-Labels <string>] [-Since <string>] [-Sort <string>] [-Direction <string>]
 [-MaxPages <uint>] [-All] [-Repo <string>] [-Forge <string>] [<CommonParameters>]
```

## ALIASES

## DESCRIPTION


Retrieves issues from GitHub or GitLab. Supports filtering by state, assignee, author, labels, and more. Automatically detects the forge provider from the current git remote, or specify one explicitly with -Forge.
