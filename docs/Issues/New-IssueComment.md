# New-IssueComment

## SYNOPSIS

Add a comment to an issue

## SYNTAX

### __AllParameterSets

```
New-IssueComment [-Id] <string> -Body <string> [-Repo <string>] [-Forge <string>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## ALIASES

## DESCRIPTION


Adds a comment to an existing issue on GitHub or GitLab.
