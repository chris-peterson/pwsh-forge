# New-ChangeRequest

## SYNOPSIS

Create a change request (pull request / merge request)

## SYNTAX

### __AllParameterSets

```
New-ChangeRequest [-Title] <string> [-SourceBranch] <string> [[-TargetBranch] <string>]
 [[-Description] <string>] [[-Repo] <string>] [[-Forge] <string>] [-Draft] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## ALIASES

## DESCRIPTION


Creates a new pull request on GitHub or merge request on GitLab.
