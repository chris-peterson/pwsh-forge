# Update-ChangeRequest

## SYNOPSIS

Update a change request

## SYNTAX

### __AllParameterSets

```

 [-TargetBranch <string>] [-Draft] [-MarkReady] [-Repo <string>] [-Forge <string>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
