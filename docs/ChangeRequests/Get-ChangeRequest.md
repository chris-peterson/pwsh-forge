# Get-ChangeRequest

## SYNOPSIS

List or get change requests (pull requests / merge requests)

## SYNTAX

### __AllParameterSets

```
Get-ChangeRequest [[-Id] <string>] [-State <string>] [-Mine] [-Group <string>]
 [-SourceBranch <string>] [-TargetBranch <string>] [-Author <string>] [-IsDraft] [-Since <string>]
 [-Until <string>] [-Reviewer <string>] [-MaxPages <uint>] [-All] [-Repo <string>] [-Forge <string>]
 [<CommonParameters>]
```

## ALIASES

## DESCRIPTION


Retrieves change requests from GitHub (pull requests) or GitLab (merge requests). Supports filtering by state, author, branches, and more.
