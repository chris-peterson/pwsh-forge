# Add-GroupMember

## SYNOPSIS

Add a member to a group or organization

## SYNTAX

### __AllParameterSets

```
Add-GroupMember [-Username] <string> -Group <string> [-Role <string>] [-Forge <string>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## ALIASES

## DESCRIPTION


Adds a user to an organization on GitHub or group on GitLab.
