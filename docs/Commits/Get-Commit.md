# Get-Commit

## SYNOPSIS

Get commit information

## SYNTAX

### __AllParameterSets

```
Get-Commit [[-Ref] <string>] [-Branch <string>] [-Author <string>] [-Since <string>]
 [-Until <string>] [-MaxPages <uint>] [-All] [-Repo <string>] [-Forge <string>] [<CommonParameters>]
```

## ALIASES

## DESCRIPTION


Retrieves commit information from a repository on GitHub or GitLab.
