# Get-UserActivity

## SYNOPSIS

Get user activity events

## SYNTAX

### __AllParameterSets

```
Get-UserActivity [[-Username] <string>] [[-Since] <string>] [[-Until] <string>] [[-Action] <string>]
 [[-TargetType] <string>] [[-Branch] <string>] [[-MaxPages] <uint>] [[-Forge] <string>] [-Mine]
 [-ExcludeMerges] [-All] [<CommonParameters>]
```

## ALIASES

## DESCRIPTION


Retrieves activity events for a user on GitHub or GitLab. Supports filtering by date range, action type, and target type.
