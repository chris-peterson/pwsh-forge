# Get-Repo

## SYNOPSIS

Get repository or project information

## SYNTAX

### __AllParameterSets

```
Get-Repo [[-Id] <string>] [-Mine] [-Group <string>] [-Select <string>] [-IncludeArchived]
 [-MaxPages <uint>] [-All] [-Forge <string>] [<CommonParameters>]
```

## ALIASES

## DESCRIPTION


Retrieves repository information from GitHub or project information from GitLab.
