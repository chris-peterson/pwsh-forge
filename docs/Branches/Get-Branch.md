# Get-Branch

## SYNOPSIS

List or get branches

## SYNTAX

### __AllParameterSets

```
Get-Branch [[-Name] <string>] [-Protected] [-Search <string>] [-MaxPages <uint>] [-All]
 [-Repo <string>] [-Forge <string>] [<CommonParameters>]
```

## ALIASES

## DESCRIPTION


Retrieves branches from a repository on GitHub or GitLab.
